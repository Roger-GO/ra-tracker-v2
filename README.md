# RA Tracker

Activity & Habit Tracker App built with Node.js, Express, and SQLite.

## Features

- Track daily activities and habits
- NLP-powered task capture
- Dashboard with analytics
- REST API

## Setup

```bash
npm install
cp .env.example .env
npm run migrate
npm start
```

## Tech Stack

- **Backend**: Node.js, Express
- **Database**: SQLite
- **NLP**: Custom parser with chrono-node
- **API**: RESTful endpoints
